Modèle de Scoring Crédit

Ce projet consistait à développer un modèle de machine learning robuste pour évaluer la solvabilité des clients et optimiser les décisions d'octroi de crédit.

Objectif

L'objectif principal était de minimiser le risque de défaut tout en maximisant les opportunités de prêt pour des clients fiables, en utilisant des techniques avancées d'analyse de données.

Technologies Utilisées

Python

SQL

Scikit-learn

Machine Learning

Démarche

  • Collecte et nettoyage de données financières hétérogènes.
  • Ingénierie de fonctionnalités pour créer des variables prédictives pertinentes.
  • Comparaison de divers algorithmes de classification (régression logistique, forêts aléatoires, boosting de gradient).
  • Optimisation des hyperparamètres et validation croisée pour assurer la robustesse du modèle.
  • Interprétabilité du modèle avec SHAP et LIME pour comprendre les facteurs de risque.

Résultats

Le modèle final a démontré une précision de 85% et un score F1 de 0.78 sur l'ensemble de test, surpassant les modèles traditionnels. Une interface simple a été développée pour permettre aux analystes de crédit d'utiliser le modèle facilement.
